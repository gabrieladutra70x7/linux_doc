# Relatório de Instalação Linux

## Projeto: Administração de Sistemas
- **0S:** Ubuntu 22.04 LTS
- **Virtualizador** Oracle VirtualBox
1. ### Lista de comandos mais usados 
| Comando | Função |
| :--- | :--- |
| `pwd` | Exibe o diretório atual (caminho) |
| `cd` | Altera o diretório de trabalho |
| `ls` | Mostra os arquivos e pastas |
| `mkdir` | Cria uma pasta (diretório) |
| `touch` | Cria um arquivo vazio ou atualiza a data de modificação de um arquivo existente |
| `cp` | Copia arquivos ou pastas |
| `mv`| Move ou renomeia arquivos ou pastas |
| `rm`| Remove arquivos |
|`nano`| Editor simples |
|`vim`| Editor complexo |
